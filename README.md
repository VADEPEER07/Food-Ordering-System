# Food-Ordering-System

Web-based food ordering application with frontend + backend.

## Features
- Register/Login
- View menu, add to cart
- Place orders
- Admin panel

## Tech Stack
- Java (Servlets/JDBC)
- MySQL
- HTML/CSS/JavaScript

## How to Run
1. Import `database.sql` into MySQL.
2. Launch backend using Java + JDBC.
3. Open `index.html` in your browser.
